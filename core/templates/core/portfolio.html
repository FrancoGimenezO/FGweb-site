PORTAFOLIO

Sitio Web Personal Desarrollado con Django

Desarrollé un sitio web personal utilizando Django, un framework web de Python, para construir una aplicación robusta y escalable. Utilicé HTML y CSS para diseñar una interfaz de usuario atractiva y responsive. El sitio consta de varias secciones, incluyendo una página de inicio dinámica, un perfil detallado en la página "Sobre Mí", un portafolio interactivo para mostrar proyectos anteriores y una página de contacto para facilitar la comunicación.

Además, implementé un sistema de control de versiones utilizando Git y GitHub. Este proyecto se encuentra alojado en mi perfil de GitHub, donde utilicé Git para gestionar eficientemente el código fuente. A través del control de versiones, pude realizar seguimiento de los cambios, revertir modificaciones si fuera necesario y trabajar en nuevas funcionalidades de manera colaborativa.

Puedes explorar el código fuente y los commits del proyecto en mi repositorio de GitHub: https://github.com/FrancoGimenezO/FGweb-site.git
